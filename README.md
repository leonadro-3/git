# git
## git 공부

깃 정리 (로컬)

1. 깃을 설치하면 로컬에서 돌아간다.
2. 로컬에서 .git init를 하여 초기 값을 설정한다.
3. git config --global user.name
4. git config --glboal user.email
5. git help tutorial
6. git help log
7. git config --list
8. echo > "example.txt" (q로 탈출)
9. del "파일" (삭제)
10. git status
11. git commit
12. git diff --cached
13. git show (커밋 해쉬값)
14. git reset --hard HEAD^
15. git checkout (커밋 해쉬값)
16. git log
17. git branch
18. git switch "master"
